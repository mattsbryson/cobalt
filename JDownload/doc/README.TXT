About
-----

JDownload is a web component software, allowing download of multiple files
and folders in a hierarchical structure to the user's local file system in
one step. No installation of download managers is required.

The official project website can be reached under the address
http://jdownload.jupload.biz/


System Requirements
-------------------

Client side:
 Java 2 Plugin, Version 1.4 or greater
 compatible browser
 write permission
 internet connection
 
Server side:
 N/A


Installation and Configuration
------------------------------

Please see the JDownload Manual in the doc/ folder for details
on installation and configuration.

Quick installation:
 1) Unzip the release archive (jdownload-xxxxx.zip) into your web folder
 2) Point your browser to the corresponding URL

Quick configuration:
 1) Edit the index.html file and point the dataURL parameter to another URL
 2) Edit the download.xml file and insert the files you want the user to download.
